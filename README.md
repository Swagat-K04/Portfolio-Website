# 🚀 Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and experience.

## 🔗 Live Demo
👉 https://swagat-portfolio-website.vercel.app/

## 📌 Features
- Clean and responsive UI
- Project showcase with detailed descriptions
- Skills and experience sections
- Easy to navigate layout
- Optimized performance with Next.js

## 🛠️ Tech Stack
- Next.js
- React
- JavaScript
- CSS

## 📂 Project Structure
```bash
src/
  app/
    components (Hero, Projects, Skills, etc.)
  data/
    portfolio.js
```

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Swagat-K04/portfolio-website.git
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:
```
http://localhost:3000
```

## 🚀 Deployment

Deployed using Vercel.

To deploy:
- Push code to GitHub
- Import repo in Vercel
- Deploy in one click
